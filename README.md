# Courier Management System

A Spring Boot REST API project for managing courier operations, including shipments, customers, warehouses, delivery agents, packages, payments, and tracking history.

## Technologies Used

- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- PostgreSQL / Oracle
- Maven
- Lombok
- REST API

## Features

- CRUD Operations
- Shipment Tracking
- Payment Management
- Warehouse & Delivery Agent Assignment
- Pagination
- Exception Handling
- Input Validation
